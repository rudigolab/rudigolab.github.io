WEB DEVELOPMENT

Front End

A hands-on creative with more than 16 years design experience.

The goal of the Front End Web Developer program is to equip learners with the unique skills they need to build and develop a variety of websites and applications. Graduates of this program will be able to construct responsive websites using CSS, Flexbox and CSS Grid, develop interactive websites and UI (User Interface) applications using JavaScript and HTML, and connect a web application to backend server data using JavaScript.

Course duration

8 months

Learning mode(s)

Hybrid

Course Outline

1. Basics of HTML and CSS to build modern day responsive websites.

2. Basics of modern ES6 JavaScript to add functionality to your websites i.e. async/await, modules, ES6 array and string methods.

3. Basics of ReactJS (JavaScript library).
